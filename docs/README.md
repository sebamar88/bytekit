### Auto-Generated Docs

Generate comprehensive API documentation:

```bash
# Generate API docs from TypeScript source
pnpm run docs:generate

# Watch for changes and regenerate
pnpm run docs:watch

# Serve docs locally on http://localhost:8080
pnpm run docs:serve
```

This creates detailed documentation including:

- Complete function signatures
- Parameter descriptions
- Return types
- JSDoc comments
- Usage examples
- Cross-references
